## Installation

1. Clone the repository :

```bash
git clone https://github.com/dewabagas/payuung-clone.git
cd klontong-app
```
2. Install dependencies :
```bash 
flutter pub get 
```
2. Run the app : 
```bash 
flutter run 
```
